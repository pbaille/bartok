# bartok

A music clojure library.

## Usage

### Litterals

alterations :bb :b :# :x

natural-pitch-classes :a :b :c :d :e :f :g

directions :u :d

pitch-class
  example: :C# 

pitch
  example: :C#u1 (u => direction , 1 => octave offset)
